Buttons

Buttons can be added in a form, search bar or to perform any simple action which makes a website interactive.

Basic buttons

Basic buttons are mostly the ones, we see and use in everyday task. These are the most common buttons. Which includes primary, secondary, outline or disabled button.

Floating buttons

Floating buttons are used to perform the most common action on the screen and they are mostly available on all the screens.

Icon buttons

Any icon can be used to act like a button

Icon text buttons

Link buttons

Buttons can also be used with anchor tags, links or inputs to trigger a functionality.
